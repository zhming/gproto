### GoProto

> protobuf 测试网站，方便测试protobuf编解码.

1. #### 技术栈
- 服务端 golang
- protobuf处理 python
- 前端 Vue3.js
- [网站地址 https://gproto.cn](https://gproto.cn)

2. #### 需求
- proto文件转python文件，存储到配置路径
- 查看proto的json完整结构
    - 处理基本类型
    - 处理对象
    - 处理嵌套对象
    - 处理数组
- protobuf byte数组数据转json结构
- json结构转protobuf byte数组
- base64编解码
